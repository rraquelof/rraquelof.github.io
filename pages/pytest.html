Como funciona?

Para desenvolver um código de qualquer tipo de aplicação é necessário que testes sejam feitos, porém quando o desenvolvedor faz seus próprios testes o instinto é criar testes em que o código vai funcionar e é por isso que existem frameworks especializados nisso, sendo um dos mais utilizados o Pytest. Ao escrever o comando que chama o pytest, ele vai analisar todos os arquivos que contenham "test.py" e vai verificar quantos e quais testes deram certo e quais deram errado.


Definições básicas

  • Nomenclatura: Para identificar os arquivos que o usuário quer que o pytest análise é necessário colocar "test_exemplo.py".
  • Install Pytest: Antes de iniciar o código com o pytest é necessário instalar a extensão.
  • Assert: O comando permite ao usuário verificar se o código funcionará nas condições propostas.
  • Shell: O pytest é executado em uma área chamada Shell, onde é necessário chamar o comando "pytest"
